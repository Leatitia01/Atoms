# **how to share some files through a network**

create a folder and put all the files in the folder

then right click on the folder 

in the dropdown menu click on properties

once the properties opens

click on the sharing tab

network file and folders sharing options will appear

click on share

network access window will open

select the account that can have access to the share

and add to permission window

click on the share option the selected folder will be share

with it content
